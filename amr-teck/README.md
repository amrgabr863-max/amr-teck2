# AMR-TECK

موقع شركة AMR-TECK المتخصصة في تصميم وتطوير المواقع الإلكترونية والحلول الرقمية.

## التقنيات المستخدمة

- React 18
- Vite
- React Router DOM
- React Helmet Async (SEO)
- Lucide React (Icons)

## طريقة التشغيل

```bash
# تثبيت الحزم
npm install

# تشغيل في وضع التطوير
npm run dev

# بناء للإنتاج
npm run build
```
